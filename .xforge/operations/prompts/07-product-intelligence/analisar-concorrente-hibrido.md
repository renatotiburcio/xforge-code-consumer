# Prompt — Analisar Concorrente Híbrido

Use `/analisar-concorrente-hibrido`.

## Objetivo

Combinar fontes locais e fontes web para criar uma visão competitiva profunda e transformar essa análise em produto, arquitetura, MVP, roadmap e backlog.

## Fontes locais

- documentos;
- PDFs;
- planilhas;
- prints;
- repositórios;
- código fonte;
- memórias anteriores;
- agentes/skills antigos;
- documentação interna.

## Fontes web

- sites oficiais;
- documentação pública;
- GitHub;
- páginas de produto;
- páginas de suporte;
- changelogs;
- materiais técnicos.

## Saídas obrigatórias

- matriz competitiva;
- matriz de features;
- mapa de módulos;
- mapa de telas;
- pontos fortes;
- pontos fracos;
- gaps;
- diferenciais recomendados;
- proposta de MVP;
- roadmap;
- backlog;
- arquitetura sugerida;
- memória atualizada;
- fontes rastreadas.
