# Contabil - Visao Geral

## Obrigacoes principais
- **ECD** (Escrituracao Contabil Digital): anual, obrigatorio para grandes empresas
- **ECF** (Escrituracao Contabil Fiscal): anual, substitui a DIPJ
- **SPED Contabil**: parte do projeto SPED
- **DRE**: Demonstracao do Resultado do Exercicio
- **Balanco Patrimonial**: Ativo, Passivo, Patrimonio Liquido

## Plano de Contas
Estrutura basica:
- 1.x.x.x - Ativo
- 2.x.x.x - Passivo
- 3.x.x.x - Patrimonio Liquido
- 4.x.x.x - Receitas
- 5.x.x.x - Custos
- 6.x.x.x - Despesas
