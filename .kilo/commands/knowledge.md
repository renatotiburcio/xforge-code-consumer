# /knowledge

Gerencie conhecimento.

## Sintaxe

```
/knowledge search
/knowledge ingest
/knowledge curate
/knowledge trust
```

## Sub-comandos

| Sub-comando | Descricao | Substitui |
|-------------|-----------|-----------|
| search | Buscar conhecimento | buscar, conhecer |
| ingest | Ingerir conhecimento | knowledge-ingest-inbox, study-documents |
| curate | Curadoria | knowledge-curation-analysis, knowledge-trust-analysis |
| trust | Score de confianca | knowledge-trust-analysis |
| web | Estudar fontes web | knowledge-study-web |
