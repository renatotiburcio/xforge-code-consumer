Wizard /forge - Exemplos de Codigo Completo

O que e

Esta pagina complementa /forge com os exemplos de codigo gerados pelos --patterns e --integrations. Use como referencia ao revisar o scaffold criado. Cobre: Program.cs, entidades DDD, CQRS Commands/Queries/Handlers, Stripe, Hangfire, SignalR, Identity, Global Error Handling.

Quando usar

  • Quando voce quer entender o codigo gerado por cada flag --patterns / --integrations
  • Quando precisa revisar o scaffold antes de commitar
  • Quando quer copiar trechos para integrar em projeto legado

Como usar

Cada cenario abaixo mostra o codigo gerado para uma flag especifica. Combine-os conforme seu wizard: /forge new MeuProjeto --auth identity --patterns ddd cqrs mediator automapper fluentvalidation result-pattern --integrations stripe hangfire sendgrid signalr.

Parametros (referencia)

ParametroDescricao
--auth identityAuth padrao (ASP.NET Core Identity)
--patternsddd cqrs mediator automapper fluentvalidation result-pattern ddd-aggregates
--integrationsstripe hangfire sendgrid signalr redis rabbitmq aws-s3
--error-handling global-middlewareProblemDetails RFC 7807 + custom handler

Exemplos praticos

Cenario 1: Program.cs completo (todos os patterns + integrations)

using MeuProjeto.Application;
using MeuProjeto.Infrastructure;
using MeuProjeto.Infrastructure.Identity;
using Microsoft.AspNetCore.Identity;
using Microsoft.EntityFrameworkCore;
using Stripe;

var builder = WebApplication.CreateBuilder(args);

// 1. DATABASE + IDENTITY (ASP.NET Core Identity - mais seguro)
builder.Services.AddDbContext<AppDbContext>(o =>
    o.UseNpgsql(builder.Configuration.GetConnectionString("Default")));
builder.Services.AddIdentity<ApplicationUser, ApplicationRole>(o =>
{
    o.Password.RequireDigit = true;
    o.Password.RequireLowercase = true;
    o.Password.RequireUppercase = true;
    o.Password.RequireNonAlphanumeric = true;
    o.Password.RequiredLength = 12;
    o.Lockout.DefaultLockoutTimeSpan = TimeSpan.FromMinutes(15);
    o.Lockout.MaxFailedAccessAttempts = 5;
    o.User.RequireUniqueEmail = true;
    o.SignIn.RequireConfirmedEmail = true;
}).AddEntityFrameworkStores<AppDbContext>().AddDefaultTokenProviders();

// 2. AUTHENTICATION (JWT + Identity)
builder.Services.AddAuthentication(o =>
{
    o.DefaultAuthenticateScheme = JwtBearerDefaults.AuthenticationScheme;
    o.DefaultChallengeScheme = JwtBearerDefaults.AuthenticationScheme;
}).AddJwtBearer(o =>
{
    o.TokenValidationParameters = new TokenValidationParameters
    {
        ValidateIssuer = true, ValidateAudience = true, ValidateLifetime = true,
        ValidateIssuerSigningKey = true,
        ValidIssuer = builder.Configuration["Jwt:Issuer"],
        ValidAudience = builder.Configuration["Jwt:Audience"],
        IssuerSigningKey = new SymmetricSecurityKey(
            Encoding.UTF8.GetBytes(builder.Configuration["Jwt:Secret"])),
        ClockSkew = TimeSpan.Zero
    };
});

// 3. MEDIATR (CQRS)
builder.Services.AddMediatR(c => c.RegisterServicesFromAssemblies(
    typeof(ApplicationLayer).Assembly, typeof(InfrastructureLayer).Assembly));
builder.Services.AddTransient(typeof(IPipelineBehavior<,>), typeof(ValidationBehavior<,>));

// 4. AUTOMAPPER
builder.Services.AddAutoMapper(typeof(MappingProfile).Assembly);

// 5. FLUENTVALIDATION
builder.Services.AddFluentValidation(o =>
    o.RegisterValidatorsFromAssemblyContaining<MappingProfile>());

// 6. STRIPE
builder.Services.AddSingleton(new StripeClient(builder.Configuration["Stripe:SecretKey"]));
builder.Services.AddScoped<IPaymentService, StripePaymentService>();

// 7. HANGFIRE
builder.Services.AddHangfire(c => c.UsePostgreSqlStorage(
    builder.Configuration.GetConnectionString("Default")));
builder.Services.AddHangfireServer();

// 8. SIGNALR
builder.Services.AddSignalR();

// 9. GLOBAL ERROR HANDLING
builder.Services.AddProblemDetails();
builder.Services.AddExceptionHandler<GlobalExceptionHandler>();

var app = builder.Build();

// PIPELINE
app.UseExceptionHandler(); // global-middleware
app.UseStatusCodePages();
if (app.Environment.IsDevelopment()) { app.UseDeveloperExceptionPage(); app.UseMigrationsEndPoint(); }
else app.UseHsts();
app.UseHttpsRedirection(); app.UseStaticFiles(); app.UseRouting();
app.UseAuthentication(); app.UseAuthorization();

// ENDPOINTS
app.MapControllers();
app.MapHub<NotificationHub>("/hubs/notifications");
app.MapHangfireDashboard("/jobs");
app.MapRazorPages(); // /Identity/Account/Login

// MIGRATIONS + SEED
using (var scope = app.Services.CreateScope())
{
    var db = scope.ServiceProvider.GetRequiredService<AppDbContext>();
    db.Database.Migrate();
    await IdentitySeeder.SeedAsync(scope.ServiceProvider);
}

app.Run();

Cenario 2: Entidades DDD com AggregateRoot (--patterns ddd-aggregates)

// src/MeuProjeto.Domain/Common/AggregateRoot.cs
public abstract class AggregateRoot<TId>
{
    public TId Id { get; protected set; }
    private readonly List<IDomainEvent> _events = new();
    public IReadOnlyList<IDomainEvent> Events => _events.AsReadOnly();
    protected void RaiseDomainEvent(IDomainEvent e) => _events.Add(e);
    public void ClearEvents() => _events.Clear();
}

// src/MeuProjeto.Domain/Common/ValueObject.cs
public abstract class ValueObject
{
    protected abstract IEnumerable<object> GetEqualityComponents();
    public override bool Equals(object obj) { /* structural eq */ }
    public override int GetHashCode() { /* hash from components */ }
}

// src/MeuProjeto.Domain/ValueObjects/Email.cs
public class Email : ValueObject
{
    public string Endereco { get; }
    private Email(string e) => Endereco = e;
    public static Email Create(string e)
    {
        if (string.IsNullOrWhiteSpace(e) || !e.Contains("@"))
            throw new DomainException("Email invalido");
        return new Email(e);
    }
    protected override IEnumerable<object> GetEqualityComponents() { yield return Endereco; }
}

// src/MeuProjeto.Domain/Aggregates/ProdutoAggregate.cs
public class ProdutoAggregate : AggregateRoot<Guid>
{
    public string Nome { get; private set; }
    public decimal Preco { get; private set; }
    public int Estoque { get; private set; }
    private ProdutoAggregate() { }
    public static ProdutoAggregate Criar(string nome, decimal preco)
    {
        var p = new ProdutoAggregate();
        p.Nome = nome; p.Preco = preco; p.Estoque = 0;
        p.RaiseDomainEvent(new ProdutoCriado(p.Id, nome));
        return p;
    }
    public void AtualizarPreco(decimal novo)
    {
        if (novo <= 0) throw new DomainException("Preco invalido");
        var old = Preco; Preco = novo;
        RaiseDomainEvent(new PrecoAlterado(Id, old, novo));
    }
}

// src/MeuProjeto.Application/Common/Result.cs (Result Pattern)
public class Result
{
    public bool IsSuccess { get; }
    public Error Error { get; }
    protected Result(bool ok, Error e) { IsSuccess = ok; Error = e; }
    public static Result Success() => new(true, Error.None);
    public static Result Failure(Error e) => new(false, e);
    public static Result<T> Success<T>(T v) => new(v, true, Error.None);
    public static Result<T> Failure<T>(Error e) => new(default, false, e);
}
public class Result<T> : Result
{
    public T Value { get; }
    protected internal Result(T? v, bool ok, Error e) : base(ok, e) { Value = v; }
}

// src/MeuProjeto.Application/Common/Error.cs
public sealed record Error(string Code, string Description, ErrorType Type = ErrorType.Failure)
{
    public static readonly Error None = new("", "", ErrorType.None);
    public static Error NotFound(string d) => new("NotFound", d, ErrorType.NotFound);
    public static Error Validation(string d) => new("Validation", d, ErrorType.Validation);
    public static Error Conflict(string d) => new("Conflict", d, ErrorType.Conflict);
}
public enum ErrorType { None, Failure, Validation, NotFound, Conflict }

Cenario 3: CQRS com MediatR (Command + Handler + Validator + Controller)

// src/MeuProjeto.Application/Commands/CreateProdutoCommand.cs
public record CreateProdutoCommand(string Nome, decimal Preco) : IRequest<Result<Guid>>;

// src/MeuProjeto.Application/Handlers/CreateProdutoHandler.cs
public class CreateProdutoHandler : IRequestHandler<CreateProdutoCommand, Result<Guid>>
{
    private readonly IProdutoRepository _repo;
    private readonly IUnitOfWork _uow;
    public CreateProdutoHandler(IProdutoRepository r, IUnitOfWork u) { _repo = r; _uow = u; }
    public async Task<Result<Guid>> Handle(CreateProdutoCommand cmd, CancellationToken ct)
    {
        var produto = ProdutoAggregate.Criar(cmd.Nome, cmd.Preco);
        await _repo.AddAsync(produto, ct);
        await _uow.SaveChangesAsync(ct);
        return Result.Success(produto.Id);
    }
}

// src/MeuProjeto.Application/Validators/CreateProdutoValidator.cs
public class CreateProdutoValidator : AbstractValidator<CreateProdutoCommand>
{
    public CreateProdutoValidator()
    {
        RuleFor(x => x.Nome).NotEmpty().MaximumLength(200);
        RuleFor(x => x.Preco).GreaterThan(0).LessThan(1_000_000);
    }
}

// src/MeuProjeto.Api/Controllers/ProdutosController.cs
[ApiController]
[Route("api/produtos")]
[Authorize]
public class ProdutosController : ControllerBase
{
    private readonly IMediator _mediator;
    public ProdutosController(IMediator m) => _mediator = m;

    [HttpPost]
    public async Task<ActionResult<Guid>> Create(CreateProdutoCommand cmd, CancellationToken ct)
    {
        var r = await _mediator.Send(cmd, ct);
        if (r.IsSuccess) return CreatedAtAction(nameof(GetById), new { id = r.Value }, r.Value);
        return r.Error.Type switch
        {
            ErrorType.Validation => ValidationProblem(...),
            _ => Problem(r.Error.Description, statusCode: 400)
        };
    }

    [HttpGet("{id}")]
    public async Task<ActionResult<ProdutoDto>> GetById(Guid id, CancellationToken ct)
    {
        var r = await _mediator.Send(new GetProdutoByIdQuery(id), ct);
        return r.IsSuccess ? Ok(r.Value) : NotFound(r.Error.Description);
    }
}

Cenario 4: Stripe PaymentService + Webhook (--integrations stripe)

// src/MeuProjeto.Application/Interfaces/IPaymentService.cs
public interface IPaymentService
{
    Task<Result<PaymentIntent>> CreatePaymentIntentAsync(decimal amount, string currency, Guid customerId, CancellationToken ct);
    Task<Result<PaymentResult>> ConfirmPaymentAsync(string paymentIntentId, CancellationToken ct);
    Task<Result<RefundResult>> RefundAsync(string chargeId, decimal amount, CancellationToken ct);
}

// src/MeuProjeto.Infrastructure/Payments/StripePaymentService.cs
public class StripePaymentService : IPaymentService
{
    private readonly StripeClient _stripe;
    private readonly ILogger<StripePaymentService> _logger;
    public StripePaymentService(StripeClient s, ILogger<StripePaymentService> l) { _stripe = s; _logger = l; }

    public async Task<Result<PaymentIntent>> CreatePaymentIntentAsync(decimal amount, string currency, Guid customerId, CancellationToken ct)
    {
        try
        {
            var intent = await _stripe.V1.PaymentIntents.CreateAsync(new PaymentIntentCreateOptions
            {
                Amount = (long)(amount * 100),
                Currency = currency,
                Metadata = new() { ["customer_id"] = customerId.ToString() }
            }, cancellationToken: ct);
            return Result.Success(new PaymentIntent(intent.Id, intent.ClientSecret));
        }
        catch (StripeException ex)
        {
            _logger.LogError(ex, "Stripe error");
            return Result.Failure(Error.Conflict($"Stripe error: {ex.StripeError?.Message}"));
        }
    }
}

// src/MeuProjeto.Api/Controllers/WebhookController.cs
[ApiController]
[Route("api/webhooks/stripe")]
public class WebhookController : ControllerBase
{
    private readonly IPaymentService _payments;
    private readonly string _webhookSecret;
    public WebhookController(IPaymentService p, IConfiguration c) { _payments = p; _webhookSecret = c["Stripe:WebhookSecret"]; }

    [HttpPost]
    public async Task<IActionResult> Handle()
    {
        var json = await new StreamReader(HttpContext.Request.Body).ReadToEndAsync();
        var stripeEvent = EventUtility.ConstructEvent(json, Request.Headers["Stripe-Signature"], _webhookSecret);
        switch (stripeEvent.Type)
        {
            case Events.PaymentIntentSucceeded:
                var intent = (PaymentIntent)stripeEvent.Data.Object;
                await _payments.ConfirmPaymentAsync(intent.Id);
                break;
            case Events.PaymentIntentPaymentFailed: /* log + notify */ break;
        }
        return Ok();
    }
}

Cenario 5: Hangfire Jobs (--integrations hangfire)

// src/MeuProjeto.Infrastructure/Jobs/Recurring/ProcessPaymentsJob.cs
public class ProcessPaymentsJob
{
    private readonly IPaymentService _payments;
    private readonly ILogger<ProcessPaymentsJob> _logger;
    public ProcessPaymentsJob(IPaymentService p, ILogger<ProcessPaymentsJob> l) { _payments = p; _logger = l; }

    [AutomaticRetry(Attempts = 3, DelaysInSeconds = new[] { 60, 300, 900 })]
    [DisableConcurrentExecution(timeoutInSeconds: 600)]
    public async Task ProcessPendingAsync(PerformContext ctx, CancellationToken ct)
    {
        _logger.LogInformation("ProcessPaymentsJob started at {Time}", DateTime.UtcNow);
        foreach (var p in await _payments.GetPendingPaymentsAsync(ct))
            await _payments.ConfirmPaymentAsync(p.IntentId, ct);
    }
}

// Program.cs: registrar jobs recorrentes
using (var scope = app.Services.CreateScope())
{
    var recurring = scope.ServiceProvider.GetRequiredService<IRecurringJobManager>();
    recurring.AddOrUpdate<ProcessPaymentsJob>("process-payments",
        j => j.ProcessPendingAsync(default, CancellationToken.None), "0 2 * * *"); // 2 AM daily
}

Cenario 6: ApplicationUser + IdentitySeeder (--auth identity)

// src/MeuProjeto.Infrastructure/Identity/ApplicationUser.cs
public class ApplicationUser : IdentityUser<Guid>
{
    public string NomeCompleto { get; set; } = "";
    public bool ConsentimentoLGPD { get; set; }
    public DateTime? DataConsentimento { get; set; }
    public string TenantId { get; set; } = ""; // multi-tenant
}
public class ApplicationRole : IdentityRole<Guid> { }

// src/MeuProjeto.Infrastructure/Identity/IdentitySeeder.cs
public static class IdentitySeeder
{
    public static async Task SeedAsync(IServiceProvider services)
    {
        using var scope = services.CreateScope();
        var userManager = scope.ServiceProvider.GetRequiredService<UserManager<ApplicationUser>>();
        var roleManager = scope.ServiceProvider.GetRequiredService<RoleManager<ApplicationRole>>();
        var config = scope.ServiceProvider.GetRequiredService<IConfiguration>();
        var adminEmail = config["Seed:AdminEmail"] ?? "admin@meuprojeto.com";
        var adminPassword = config["Seed:AdminPassword"] ?? "ChangeMe!2026";

        string[] roles = { "Admin", "Manager", "User" };
        foreach (var r in roles) if (!await roleManager.RoleExistsAsync(r)) await roleManager.CreateAsync(new ApplicationRole { Name = r });

        if (await userManager.FindByEmailAsync(adminEmail) == null)
        {
            var admin = new ApplicationUser { UserName = adminEmail, Email = adminEmail, NomeCompleto = "Admin", ConsentimentoLGPD = true, DataConsentimento = DateTime.UtcNow };
            await userManager.CreateAsync(admin, adminPassword);
            await userManager.AddToRoleAsync(admin, "Admin");
        }
    }
}

Troubleshooting

Identity migration falha

Causa: connection string invalida ou Postgres inacessivel.

Solucao: rode dotnet ef database update manual; verifique appsettings.json.

MediatR nao registra handlers

Causa: assembly do handler nao em RegisterServicesFromAssemblies.

Solucao: adicione o assembly (Application ou Infrastructure) na chamada.

AutoMapper nao mapeia

Causa: MappingProfile nao escaneado.

Solucao: confirme assembly correto em AddAutoMapper(typeof(MappingProfile).Assembly).

FluentValidation nao dispara

Causa: validators assembly nao registrada OU pipeline behavior ausente.

Solucao: confirme RegisterValidatorsFromAssemblyContaining E AddTransient(typeof(IPipelineBehavior<,>), typeof(ValidationBehavior<,>)).

Stripe webhook signature invalida

Causa: webhook secret nao configurado.

Solucao: configure STRIPE_WEBHOOK_SECRET em appsettings.Production.json e Stripe Dashboard.

Hangfire jobs nao executam

Causa: server nao iniciado OU cron invalida.

Solucao: confirme app.UseHangfireServer() e valide cron em https://crontab.guru.
