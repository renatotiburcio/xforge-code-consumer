# Expert Router

## Objetivo

Selecionar especialistas automaticamente.

## Exemplo

Pedido: criar folha de pagamento.

Experts acionados:

- folha/payroll;
- trabalhista;
- eSocial;
- EFD-Reinf;
- contábil;
- financeiro;
- segurança/LGPD;
- .NET;
- banco;
- Blazor;
- UX;
- QA;
- documentação;
- suporte.

## Regra

O usuário não escolhe experts manualmente, exceto quando quiser.
