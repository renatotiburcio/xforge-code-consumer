# Glossario de Termos de Seguranca da Informacao

## A

- **AAA**: Authentication, Authorization, Accounting
- **ACL**: Access Control List
- **AES**: Advanced Encryption Standard (128/192/256 bits)
- **APT**: Advanced Persistent Threat
- **ASVS**: Application Security Verification Standard (OWASP)

## B

- **Bcrypt**: Hash de senhas (Blowfish variant)
- **BOLA**: Broken Object Level Authorization
- **Botnet**: Rede de dispositivos comprometidos

## C

- **CA**: Certificate Authority
- **CAPTCHA**: Teste para distinguir humanos de bots
- **CIS Controls**: 18 controles criticos
- **CSP**: Content Security Policy
- **CWE**: Common Weakness Enumeration

## D

- **DAST**: Dynamic Application Security Testing
- **DDoS**: Distributed Denial of Service
- **DKIM**: DomainKeys Identified Mail
- **DMARC**: Validacao de email
- **DNSSEC**: Seguranca DNS

## E

- **E2EE**: End-to-End Encryption
- **EDR**: Endpoint Detection and Response
- **ERP**: Enterprise Resource Planning
- **EV**: Extended Validation (SSL)

## F

- **FIDO2**: Autenticacao sem senha
- **FIPS**: Federal Information Processing Standards
- **Forensics**: Investigacao pos-incidente

## G

- **GDPR**: General Data Protection Regulation (UE)
- **GPG/PGP**: Criptografia chave publica

## H

- **HIDS**: Host-based IDS
- **HSM**: Hardware Security Module
- **HSTS**: HTTP Strict Transport Security

## I

- **IAM**: Identity and Access Management
- **IDS/IPS**: Intrusion Detection/Prevention
- **IKE**: Internet Key Exchange
- **IOC**: Indicator of Compromise
- **ISO 27001**: Norma gestao seguranca informacao

## J

- **JWT**: JSON Web Token

## K

- **Kerberos**: Autenticacao em redes (Windows AD)

## L

- **LDAP**: Lightweight Directory Access Protocol
- **LGPD**: Lei Geral de Protecao de Dados (BR)
- **LFI**: Local File Inclusion

## M

- **MAC**: Mandatory Access Control
- **MFA**: Multi-Factor Authentication
- **MITM**: Man-in-the-Middle
- **MSSP**: Managed Security Service Provider

## N

- **NAC**: Network Access Control
- **NIST**: National Institute of Standards and Technology
- **NVD**: National Vulnerability Database

## O

- **OAuth 2.0**: Autorizacao delegada
- **OWASP**: Open Web Application Security Project
- **OWASP Top 10**: Top 10 vulnerabilidades web

## P

- **PAM**: Privileged Access Management
- **PCI-DSS**: Payment Card Industry DSS
- **PII**: Personally Identifiable Information
- **PKI**: Public Key Infrastructure
- **PTES**: Penetration Testing Execution Standard

## Q

- **QSA**: Qualified Security Assessor

## R

- **RADIUS**: Autenticacao centralizada
- **RBAC**: Role-Based Access Control
- **RPO**: Recovery Point Objective
- **RSA**: Criptografia assimetrica
- **RTO**: Recovery Time Objective

## S

- **SAML**: Security Assertion Markup Language
- **SAST**: Static Application Security Testing
- **SCADA**: Controle industrial
- **SIEM**: Security Information and Event Management
- **SOAR**: Security Orchestration, Automation and Response
- **SOC**: Security Operations Center
- **SQLi**: SQL Injection
- **SSH**: Secure Shell
- **SSO**: Single Sign-On
- **SSL/TLS**: Criptografia em transito

## T

- **TDE**: Transparent Data Encryption
- **TLS 1.3**: Versao mais recente (2018)
- **TOCTOU**: Time of Check to Time of Use

## U

- **UEBA**: User and Entity Behavior Analytics

## V

- **VA**: Vulnerability Assessment
- **VPN**: Virtual Private Network
- **VPC**: Virtual Private Cloud
- **VS**: Verification Standard

## W

- **WAF**: Web Application Firewall
- **WPA3**: Padrao Wi-Fi mais recente

## X

- **XDR**: Extended Detection and Response
- **XSS**: Cross-Site Scripting
- **XXE**: XML External Entity

## Z

- **Zero Trust**: Confianca zero por padrao
- **ZTA**: Zero Trust Architecture
