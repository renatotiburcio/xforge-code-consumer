# Legacy Import Rules

1. Conteúdo legado nunca entra direto no Engineer atual.
2. Todo legado deve ser inventariado.
3. Todo legado deve ser classificado por tipo.
4. Todo legado deve ser comparado com o estado atual.
5. Duplicidades devem ser consolidadas.
6. Conflitos devem ser documentados.
7. Conteúdo melhor deve ser promovido somente após validação.
8. Conteúdo histórico útil deve ser arquivado.
9. Conteúdo inválido deve ser rejeitado com motivo.
