# /configurar

## Objetivo
Configurar e personalizar todo o ambiente XForge: providers de AI, GitHub, Tailwind, padrões .NET.

## Sub-comandos

| Sub-comando | Acao |
|-------------|------|
| `/configurar ai` | Configurar providers de AI (OpenRouter, Ollama, etc.) |
| `/configurar github` | Configurar GitHub Enterprise e tokens |
| `/configurar tailwind` | Configurar Tailwind CSS para Blazor/frontend |
| `/configurar dotnet` | Definir padrões .NET (Clean Architecture, CQRS, etc.) |
| `/configurar status` | Mostrar estado atual das configuracoes |

## Uso
```
/configurar ai
/configular github
/configular status
```
