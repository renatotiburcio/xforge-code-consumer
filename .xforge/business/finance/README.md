# Financeiro - Visao Geral

## Modulos
- **Contas a pagar**: fornecedores, impostos, folha
- **Contas a receber**: clientes, duplicatas
- **Conciliacao bancaria**: matching automatico
- **Fluxo de caixa**: previsao diaria/semanal/mensal
- **Tesouraria**: aplicacoes, emprestimos
- **Rateio**: centros de custo

## Cadencias
- **Diaria**: movimento bancario, baixa de titulos
- **Semanal**: pagamento programado, fluxo projetado
- **Mensal**: fechamento, aging list, fluxo realizado
- **Anual**: planejamento orcamentario
