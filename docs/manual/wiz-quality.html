Wizard /quality - Auditar Qualidade

O que e

/quality roda quality gates automaticamente: build verde, test coverage, SOLID violations, code smells, tech debt score. Sugere refactorings priorizados. Gera relatorio em human-readable, JSON, ou JUnit para CI.

Quando usar

  • Quando quer auditar qualidade do codigo antes de merge
  • Quando precisa de relatorio para stakeholders
  • Quando quer priorizar refactorings

Como usar

Sintaxe canonica:

/quality --scope tests --severity low --format human

Para invocar: abra o XForge Code CLI e digite o comando. O wizard guia via Q&A interativo.

Parametros

ParametroTipoDefaultDescricaoObrigatorio?
--scopestringalltests, coverage, solid, debt, allNao

Exemplos praticos

Cenario 1: Auditoria pre-merge (basica)

Contexto: Vai fazer PR e quer checar qualidade.

/quality --scope tests --severity low --format human

Output esperado:

[quality] 142 tests. 138 OK. 4 skipped. Coverage 87% (target 85%).
[quality] Build: 0 errors, 2 warnings.
[quality] Resultado: OK to merge.

Edge case: Coverage < 85% bloqueia merge automatico.

Cenario 2: Auditoria SOLID completa

Contexto: Codigo com suspects de violacao SOLID. Quer relatorio detalhado.

/quality --scope solid --format json --output ./solid-report.json

Output esperado:

[quality] 23 SOLID violations detectadas:
  SRP (8): OrderService (4 responsabilidades), CustomerService (3), ...
  OCP (7): Precisa de interface para extensao em 7 classes
  LSP (4): Subclasses quebram contrato em 4 hierarquias
  ISP (3): Interfaces muito grandes em 3 arquivos
  DIP (1): Dependencia direta de implementacao em 1 classe

[quality] Top priority: OrderService.cs (4 violations, 230 linhas)
[quality] Relatorio: ./solid-report.json

Edge case: Violacoes DIP sao as mais problematicas (acoplamento).

Cenario 3: Auditoria completa com baseline

Contexto: Quer comparar com ultima execucao (melhorou ou piorou?).

/quality --scope all --baseline last-run --format html

Output esperado:

[quality] 47 checks. Comparando com baseline (2026-06-20):
  -2 SOLID violations (melhorou!)
  +5% coverage (melhorou!)
  +3 tests (melhorou!)
  -1 warning (melhorou!)
  +0 critical (mesmo)

[quality] Score: 87/100 (vs 79/100 baseline). +8 pontos.
[quality] Relatorio: ./quality-report.html

Edge case: Baseline fica em .xforge/reports/quality-baseline.json.
