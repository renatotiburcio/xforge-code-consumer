## Bug Patterns Learned (10 fixes documented)

1. Shell escape parens → chr(34) concatenation
2. Module naming _common.py → renomear para common.py
3. Circular import tool_X → lazy import dentro da funcao
4. Subprocess sys.path → bootstrap pattern
5. Test state pollution → reset em _fresh_engine()
6. Module cache TTL env var → force reload explicito
7. Range removal scope → backup scan antes de remove
8. Accidental TOOLS removal → restore manual
9. Duplicate _ok/_err → remove duplicates
10. CR/LF PowerShell → Python file API direto
