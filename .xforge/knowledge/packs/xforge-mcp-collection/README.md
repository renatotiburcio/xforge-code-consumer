# xforge-mcp-collection

Pack referencia para MCPs (Model Context Protocol) integrados ao XForge.

## MCPs incluidos

- brave-search-mcp (8 search tools)
- context7-mcp (library docs)
- dbhub-mcp (multi-database)
- filesystem-mcp (12 file tools)
- firecrawl-mcp (20 web scraping tools)
- github-mcp (21 toolsets, ~100+ tools)
- linear-mcp (issue tracking)
- markitdown-mcp (PDF/DOCX/image to markdown)
- memory-mcp (cross-chat USER memory)
- mermaid-mcp (diagram generation)
- plantuml-mcp (UML/C4 diagrams)
- playwright-mcp (50+ browser automation)
- sequential-thinking-mcp (structured reasoning)
- slack-mcp (team chat)

## Quando usar

- Ao configurar MCPs no kilo.jsonc
- Ao buscar dados externos (web, DB, filesystem)
- Ao usar ferramentas especializadas via MCP
