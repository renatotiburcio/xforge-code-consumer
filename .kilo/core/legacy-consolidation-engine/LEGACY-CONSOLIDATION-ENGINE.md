# Legacy Consolidation Engine

## Objetivo

Transformar múltiplos sistemas, bibliotecas, pacotes, agentes, skills e documentações legadas em um ecossistema XForge consolidado.

## Entrada

```text
legacy/
```

Pode conter:

- ERP antigo;
- financeiro;
- investimentos;
- estoque;
- contabilidade;
- folha;
- design system;
- bibliotecas;
- pacotes;
- documentações;
- agentes;
- skills;
- memórias.

## Saída

- mapa funcional;
- duplicidades;
- conflitos;
- reaproveitamento;
- arquitetura consolidada;
- roadmap de migração;
- backlog;
- knowledge update;
- project DNA.
