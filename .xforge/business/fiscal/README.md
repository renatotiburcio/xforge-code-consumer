# Fiscal - Visao Geral

## Obrigacoes principais
- **NF-e / NFC-e**: documentos fiscais eletronicos
- **CT-e / MDF-e**: conhecimento e manifesto de transporte
- **NFS-e**: nota fiscal de servico eletronica
- **SPED**: Escrituracao Fiscal Digital
- **EFD ICMS/IPI**: escrituracao fiscal
- **EFD Contribuicoes**: PIS/COFINS
- **Reforma Tributaria**: IBS, CBS, Imposto Seletivo (2026-2033)
