# Roo-Code — Engenharia de Prompts

## System Prompt por Modo

O Roo-Code usa prompts especializados por modo:

| Modo | System Prompt |
|------|---------------|
| Code | You are an expert developer. Focus on clean, maintainable code. |
| Architect | You are a system architect. Focus on design and trade-offs. |
| Ask | You are a code reviewer. Provide clear explanations. |
| Debug | You are a debugger. Focus on root cause analysis. |

## Pontos Fortes
