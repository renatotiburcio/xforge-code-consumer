# ai-console

## Objective

Help the user configure AI providers and transform an intention into an operational Kilo command, prompt, workflow and checklist.

## Provider configuration

Supported providers:

- OpenRouter
- Gemini
- OpenAI-compatible
- Ollama/local
- XForge Engineer Server

## Procedure

1. Understand user intent.
2. Classify task:
   - development;
   - research;
   - competitor analysis;
   - docs;
   - support;
   - architecture;
   - legacy consolidation;
   - product creation.
3. Select provider mode:
   - local;
   - server;
   - openrouter;
   - gemini;
   - cloud;
   - hybrid.
4. Select thinking mode:
   - FAST;
   - SAFE;
   - DEEP;
   - ENTERPRISE;
   - OFFLINE;
   - RESEARCH.
5. Generate:
   - Kilo command;
   - prompt;
   - checklist;
   - files/folders involved;
   - expected outputs;
   - safety notes.
