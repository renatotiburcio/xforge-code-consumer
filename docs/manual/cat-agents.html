Agents - Catalogo Completo (53)

O que e

Catalogo completo de todos os 53 agents do XForge: 12 canonical (routing), 7 directors (senior), 3 managers (state), 28 genius council, e specialists. Use para descobrir qual agent atende sua necessidade.

Quando usar

  • Quando voce quer descobrir todos os itens desta categoria
  • Quando voce precisa de referencia rapida com descricao
  • Quando voce quer ver agrupamento por subcategoria

Como usar

Esta pagina cataloga todos os itens. Cada item tem:

[Nome do Item] - [Descricao curta]

Para detalhes, clique no item ou consulte o arquivo SKILL.md no repositorio.

Parametros

CampoTipoDescricao
NomestringIdentificador kebab-case en
DescricaostringResumo de 1 frase
CategoriastringSubcategoria dominante
Skill.mdpath.kilo/skills/<nome>/SKILL.md

Exemplos praticos

Cenario 1: Localizar um item especifico

Contexto: voce quer descobrir todas as skills Backend disponiveis.

# Use Ctrl+F no navegador para buscar por nome ou keyword
# Ou role ate a secao Backend abaixo

Output esperado: lista das skills Backend com descricoes e links.

Edge case: se a skill nao aparece, ela pode ter sido renomeada ou descontinuada - verifique o filesystem em .kilo/skills/.

Cenario 2: Navegar por categoria (22 itens no total)

Contexto: voce quer ver agrupamento por subcategoria.

# Role pelas secoes abaixo
# Cada secao = 1 subcategoria
# Cada item = 1 artefato

Output esperado: visualizacao hierarquica por subcategoria.

Edge case: categorias podem ter 0-30+ itens.

Cenario 3: Encontrar categoria certa

Contexto: voce sabe o que quer mas nao sabe o nome exato.

# Use Ctrl+F e busque por keyword
# Ex: "LGPD" encontra skills de LGPD/data-protection

Output esperado: matches highlighted na pagina.

Edge case: alguns itens tem descricoes vagas. Use o ID do filesystem (.kilo/skills/<nome>/SKILL.md) como busca definitiva.

Troubleshooting

Item nao aparece na lista

Causa: pode estar em categoria diferente ou ainda nao catalogada.

Solucao: verifique .kilo/skills/ diretamente.

Descricao vazia

Causa: SKILL.md sem frontmatter description:.

Solucao: edite .kilo/skills/<nome>/SKILL.md e adicione frontmatter YAML.

Categoria muito grande

Causa: subcategoria com 30+ itens.

Solucao: use Ctrl+F para buscar, ou subdivida em sub-subcategorias no futuro.

Item listado mas nao existe mais

Causa: SKILL.md deletado mas listagem nao regenerada.

Solucao: rode regenerate-registries.ps1 para sincronizar.
