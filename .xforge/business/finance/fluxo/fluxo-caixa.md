# Fluxo de Caixa

## Tipos
- **Realizado**: baixas ja confirmadas
- **Projetado**: titulos a pagar/receber com data prevista
- **Provisionado**: orcamentos nao confirmados

## DRE resumo (mensal)
```
(+) Receita Bruta
(-) Deducoes (impostos sobre venda, devolucoes)
= Receita Liquida
(-) CMV / CSP
= Lucro Bruto
(-) Despesas Operacionais
= EBITDA
(-) Depreciacao / Amortizacao
= EBIT
(-) Despesas Financeiras
= LAIR
(-) IRPJ / CSLL
= Lucro Liquido
```

## Fluxo diario
```
Saldo Inicial
(+) Recebimentos do dia
(-) Pagamentos do dia
= Saldo Final
```

## Indicadores
- **Liquidez Corrente**: AC / PC
- **Liquidez Seca**: (AC - Estoque) / PC
- **Giro do Caixa**: Receita / Saldo Medio
- **Burn Rate**: consumo mensal medio de caixa
- **Runway**: meses ate caixa zerar
