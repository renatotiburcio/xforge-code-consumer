# ERP - Visao Geral

## Modulos principais
- **Cadastros**: clientes, fornecedores, produtos
- **Compras**: pedido de compra, cotacao, ordem de compra
- **Vendas**: pedido de venda, orcamento, expedicao
- **Estoque**: entrada, saida, inventario, transferencias
- **Financeiro**: contas a pagar, contas a receber, fluxo de caixa
- **Producao**: ordem de producao, lista de materiais, roteiros
