# Brief - Personal Portfolio Landing Page

## Problema
Profissionais de tecnologia (devs, designers, arquitetos) precisam de um portfolio online simples, rapido e gratuito para divulgar seu trabalho, sem depender de plataformas SaaS pagas.

## Objetivo
Criar uma landing page pessoal com HTML + Tailwind CSS (CDN), 100% estatica, sem build, sem backend. Uma unica pagina `index.html` com inline CSS e JS minimo. Deploy via GitHub Pages, Vercel ou Netlify.

## Stack Detectado
**HTML+Tailwind standalone** (sinal: `index.html` na raiz sem build).

## Fora de Escopo
- Backend (formulario usa mailto: ou formspree)
- CMS (conteudo hardcoded em HTML)
- Multi-idioma
- Auth/admin
- Database
- SSR/SSG (estatico)

## Personas
- **Visitante anonimo**: quer ver portfolio, contratar, ou apenas navegar
- **Recrutador**: quer validar skills rapido (< 30s)

## Cenarios
- Visitante chega via Google e ve portfolio em < 2s
- Recrutador rola ate "Experiencia" e clica em um projeto
- Visitante clica em "Contato" e abre mailto
