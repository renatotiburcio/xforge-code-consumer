# XForge Dashboard

## Pages

| Page | Route | Description |
|------|-------|-------------|
| Health | `/` | Doctor results, system status |
| RAG | `/rag` | Index stats, query testing, health |
| Agents | `/agents` | Agent list, details, modes |
| Skills | `/skills` | Skill categories, search |
| Commands | `/commands` | Command catalog with examples |

## Tech Stack

- Next.js 14 (App Router)
- React 18
- Tailwind CSS
- shadcn/ui components
- Lucide icons

## Setup

```bash
npm install
npm run dev
```
