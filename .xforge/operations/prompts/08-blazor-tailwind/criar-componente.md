# Prompt — Criar Componente Blazor

Use `/criar-componente-blazor`.

## Objetivo

Criar componente Blazor enterprise usando Tailwind CSS v4.

## O componente deve possuir

- variants;
- states;
- loading;
- disabled;
- error;
- empty;
- keyboard navigation;
- accessibility;
- responsividade;
- documentação;
- exemplos.

## Regras

- utility-first;
- sem duplicação visual;
- sem CSS inline desnecessário;
- compatível com design system;
- compatível com dark/light mode;
- fácil de reutilizar em ERP.
