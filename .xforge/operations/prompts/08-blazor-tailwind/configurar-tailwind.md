# Prompt — Configurar Tailwind em Blazor

Use `/configurar-tailwind-blazor`.

## Objetivo

Configurar Tailwind CSS v4 no projeto Blazor atual com padrão enterprise XForge.

## Deve fazer

1. Detectar tipo de projeto:
   - Blazor Server;
   - Blazor WebAssembly;
   - Blazor WebApp;
   - Blazor Hybrid.

2. Configurar:
   - `tailwindcss`;
   - `@tailwindcss/cli`;
   - `input.css`;
   - `output.css`;
   - scripts de build/watch;
   - tokens;
   - themes;
   - utilities.

3. Atualizar:
   - documentação;
   - PROJECT-DNA;
   - memória frontend;
   - quality checklist.

## Regras

- não espalhar CSS;
- usar utility-first;
- design tokens;
- responsividade obrigatória;
- acessibilidade obrigatória.
