# /testar

## Objetivo
Executar todos os testes do projeto: unitarios, integracao, e2e e cobertura.

## Sub-comandos

| Sub-comando | Acao |
|-------------|------|
| `/testar unit` | Testes unitarios |
| `/testar integration` | Testes de integracao |
| `/testar e2e` | Testes end-to-end |
| `/testar coverage` | Relatorio de cobertura |
| `/testar all` | Todos os testes |

## Uso
```
/testar unit
/testar integration
/testar e2e
/testar coverage
/testar all
```
