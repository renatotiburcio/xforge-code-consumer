### Isenção Efetiva

O projeto de lei previa isenção até 5 salários mínimos (R$ 7.590,00), mas a Lei 14.848/2024 aprovou com faixa isenta de R$ 2.259,20 (1,5 salários mínimos).

**IMPORTANTE:** O governo publicou decreto com faixa isenta até R$ 5.000, mas a legislação final aprovada é a tabela acima. Verificar sempre a versão vigente.

## Fórmula de Cálculo Completa

### Passo 1: Calcular INSS
```
INSS = (Salário Bruto × Alíquota) - Parcela a Deduzir
```

### Passo 2: Base do IRRF
```
Base IRRF = Salário Bruto - INSS - Dependentes - Pensão
```

### Passo 3: Calcular IRRF
```
IRRF = (Base IRRF × Alíquota) - Parcela a Deduzir
```

### Passo 4: Salário Líquido
```
Líquido = Salário Bruto - INSS - IRRF
```

## Exemplos Completos (2025)

### Exemplo 1: Salário R$ 2.000 (ISENTO)

1. INSS: R$ 2.000 × 9% - 22,77 = **R$ 157,23**
2. Base IRRF: R$ 2.000 - 157,23 = **R$ 1.842,77**
3. Faixa IRRF: 1ª (até R$ 2.259,20) = **ISENTO**
4. Líquido: R$ 2.000 - 157,23 = **R$ 1.842,77**

### Exemplo 2: Salário R$ 3.000

1. INSS: R$ 3.000 × 12% - 106,59 = **R$ 253,41**
2. Base IRRF: R$ 3.000 - 253,41 = **R$ 2.746,59**
3. Faixa IRRF: 2ª (R$ 2.259,21 a R$ 2.826,65)
4. IRRF: R$ 2.746,59 × 7,5% - 169,44 = **R$ 36,56**
5. Líquido: R$ 3.000 - 253,41 - 36,56 = **R$ 2.710,03**

### Exemplo 3: Salário R$ 5.000

1. INSS: R$ 5.000 × 14% - 190,40 = **R$ 509,60**
2. Base IRRF: R$ 5.000 - 509,60 = **R$ 4.490,40**
3. Faixa IRRF: 4ª (R$ 3.751,06 a R$ 4.664,68)
4. IRRF: R$ 4.490,40 × 22,5% - 662,77 = **R$ 347,57**
5. Líquido: R$ 5.000 - 509,60 - 347,57 = **R$ 4.142,83**

### Exemplo 4: Salário R$ 5.000 com 2 dependentes

1. INSS: R$ 5.000 × 14% - 190,40 = **R$ 509,60**
2. Base IRRF: R$ 5.000 - 509,60 - (2 × 189,59) = **R$ 4.111,22**
3. Faixa IRRF: 4ª (R$ 3.751,06 a R$ 4.664,68)
4. IRRF: R$ 4.111,22 × 22,5% - 662,77 = **R$ 262,26**
5. Líquido: R$ 5.000 - 509,60 - 262,26 = **R$ 4.228,14**

### Exemplo 5: Salário R$ 7.000, casado, 3 dependentes, pensão R$ 800

1. INSS: R$ 7.000 × 14% - 190,40 = **R$ 789,60**
2. Base IRRF: R$ 7.000 - 789,60 - (3 × 189,59) - 800 = **R$ 4.851,83**
3. Faixa IRRF: 5ª (acima de R$ 4.664,68)
4. IRRF: R$ 4.851,83 × 27,5% - 896,00 = **R$ 438,25**
5. Líquido: R$ 7.000 - 789,60 - 438,25 = **R$ 5.772,15**
