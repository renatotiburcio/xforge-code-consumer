Cat-skills Fiscal (BR) - Impostos, NF-e, SPED

O que e

Catalogo de skills para area fiscal brasileira. Inclui: NF-e/NFC-e (emissao, recepcao, cancelamento, inutilizacao), SPED Fiscal (EFD ICMS/IPI), SPED Contribuicoes (EFD PIS/COFINS), impostos (ICMS, IPI, PIS, COFINS, ISS, IRPJ, CSLL), Reforma Tributaria (IBS, CBS), CFOP, NCM, CST, aliquotas por UF. Skills canonicas: fiscal-tax-expert, sped-fiscal-expert, nfe-expert.

Quando usar

  • Quando precisa emitir NF-e/NFC-e
  • Quando precisa configurar aliquotas de ICMS por UF
  • Quando precisa gerar SPED Fiscal mensal
  • Quando precisa lidar com Reforma Tributaria (IBS/CBS)
  • Quando precisa calcular impostos em vendas/compras

Como usar

Skills fiscais carregam tabelas oficiais atualizadas (CFOP, NCM, CST, aliquotas). Verifique TTL - legislacao muda frequentemente.

# Emitir NF-e
/nfe-emitir --serie 1 --numero 12345 --valor 1500.00 --cliente 
# Gerar SPED Fiscal do mes
/sped-fiscal --periodo 2025-06 --tipo EFD-ICMS-IPI
# Consultar tabela CFOP
/fiscal-cfop --natureza venda --estado SP

Parametros

ParametroTipoDefaultDescricao
--estadostringautoUF para aliquotas (SP, RJ, MG, etc). auto = detecta do projeto
--regimestringautoRegime tributario: simples, presumido, real, auto
--periodoYYYY-MMatualPeriodo de apuracao
--tabelastringatualTabela de referencia: CFOP, NCM, CST, aliquotas

Exemplos praticos

Cenario 1: Emitir NF-e de venda

Contexto: Voce fez uma venda B2B e precisa emitir NF-e.

/nfe-emitir --tipo venda-b2b --valor 1500.00 --cliente 12345678000190 --cfop 6101

Output esperado:

NF-e gerada com protocolo SEFAZ. XML assinado. Status: autorizada em 2.5s.

Edge case: CFOP 6101 = venda de producao propria interestadual. Para dentro do estado use 5101.

Cenario 2: Configurar ICMS por UF

Contexto: Sua empresa vende para todo Brasil. Precisa de tabela de ICMS.

/fiscal-icms-tabela --gerar --base 2025-01

Output esperado:

Tabela CSV com aliquotas ICMS intra/inter por UF (27 UFs x 27 UFs). Para 2025: SP-RJ 12%, SP-MG 7%, etc.

Edge case: Aliquotas mudam 4x ao ano (geralmente). Use TTL 3 meses para tabela ICMS (DR-ttl-rules).

Cenario 3: Gerar SPED Fiscal mensal

Contexto: Venceu o prazo. Precisa gerar EFD ICMS/IPI do mes.

/sped-fiscal --periodo 2025-06 --validar pre-validar --gerar-bloco-0

Output esperado:

Arquivo TXT do SPED Fiscal gerado (Bloco 0: cadastro, depois Bloco C: notas, etc). Pre-validacao: 0 erros, 3 warnings (verificar).

Edge case: Bloco K (industria) deve ser gerado se empresa produz. Bloco H (inventario) e obrigatorio se houver estoque.

Cenario 4: Calcular PIS/COFINS cumulativo

Contexto: Voce esta no regime cumulativo (simples ou presumido) e precisa calcular.

/fiscal-pis-cofins --regime cumulativo --base 100000.00

Output esperado:

PIS: 0.65% = R$ 650. COFINS: 3.00% = R$ 3.000. Total: R$ 3.650.

Edge case: No regime NAO-cumulativo, use aliquotas 1.65% PIS e 7.60% COFINS com creditos.

Cenario 5: Reforma Tributaria (IBS/CBS)

Contexto: Voce precisa se preparar para Reforma Tributaria (2026+).

/fiscal-reforma-tributaria --status 2025

Output esperado:

IBS (estadual/municipal) e CBS (federal) substitui ICMS/ISS/PIS/COFINS gradualmente 2026-2032. Aliquota teste: 17.7%.

Edge case: Fique atento as DRs de reforma tributaria - regras transitorias estao em publicacao constante.

Troubleshooting

NF-e rejeitada por motivo X

Causa: Rejeicoes tem motivos especificos (cStat). Verifique o codigo

Solucao: Use /fiscal-rejeicoes --cstat para ver causa e solucao. Codigos mais comuns: 102 (XML invalido), 539 (duplicado), 561 (CNPJ invalido)

Aliquota ICMS errada para UF destino

Causa: Cada UF tem aliquotas diferentes para intra e interestadual

Solucao: Use /fiscal-icms-tabela para gerar tabela atualizada. SP-RJ intra=18%, inter=12%. Norte/Nordeste/ES/MG: 7% para SP.

SPED com erros de validacao

Causa: Campos obrigatorios faltando ou formatos invalidos

Solucao: Use /sped-validar --arquivo para ver erros detalhados. Cada erro tem codigo + linha + sugestao.

NCM nao encontrado

Causa: NCM (codigo fiscal) tem 8 digitos e milhares de opcoes

Solucao: Use /fiscal-ncm-buscar --descricao . Ou consulte Tabela NCM em .xforge/knowledge/dominios/fiscal/
