# /develop

Desenvolvimento de software.

## Sintaxe

```
/develop feature
/develop fix
/develop refactor
/develop improve
```

## Sub-comandos

| Sub-comando | Descricao | Substitui |
|-------------|-----------|-----------|
| feature | Nova feature | create-feature, development-create-crud |
| fix | Correcao de bug | fix-incident, incident-analysis |
| refactor | Refatoracao | development-recognize-project |
| improve | Melhoria | development-create-anything |
