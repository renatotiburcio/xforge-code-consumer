# /create

Crie artefatos de software.

## Sintaxe

```
/create projeto
/create feature
/create api
/create componente
/create documento
/create treinamento
```

## Sub-comandos

| Sub-comando | Descricao | Substitui |
|-------------|-----------|-----------|
| projeto | Novo projeto | create-project, create-dotnet-project, create-blazor, create-erp, development-create-anything |
| feature | Nova feature | create-feature, development-create-crud |
| api | Nova API | create-api |
| componente | Novo componente | create-blazor-component |
| documento | Documentacao | create-bilingual-manual, document, documentacao |
| treinamento | Trilha de treinamento | create-training-track |
