# Agent Handoff Template

Use this template when one agent delegates to another.

```text
handoff_id:
from_agent:
to_agent:
user_goal:
current_state:
risk_level:
project_area:
files_read:
files_changed:
memory_used:
constraints:
requested_output:
validation_required:
blocking_questions:
```

## Rules

- Keep handoffs short.
- Include only relevant files.
- Always include risk level.
- Never hide unresolved uncertainty.
- Return with findings, actions, validation results and residual risks.
