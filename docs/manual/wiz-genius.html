Wizard /genius - Adicionar Genio ao Conselho

O que e

/genius implementa CREATE_GENIUS template. Coleta nome, inspiracao, missao, especialidades, perguntas caracteristicas, responsabilidades, criterios de qualidade, e dominios de participacao. Gera arquivo .kilo/agents/genius-council/AGXXX-nome.md + memoria .xforge/memory/genius/AGXXX/knowledge.jsonl + DR formal.

Quando usar

  • Quando precisa adicionar especialista novo ao Conselho
  • Quando quer documentar figura historica ou domain expert
  • Quando quer expandir memoria do Conselho com conhecimento novo

Como usar

Sintaxe canonica:

/genius --name "Marcos Oliveira" --inspiration "Auditor fiscal senior, ex-receita federal" --domain "Fiscal" --skills-count 6

Para invocar: abra o XForge Code CLI e digite o comando. O wizard guia via Q&A interativo.

Parametros

ParametroTipoDefaultDescricaoObrigatorio?
--name <nome>stringobrigatorioNome do genio (AG067, AG068, ...)Sim
--inspiration <figura>stringobrigatorioFigura historica ou domain expertSim
--domain <area>stringobrigatorioBackend, Frontend, AI, Fiscal, etcSim
--mission <texto>stringobrigatorio1 frase descrevendo missaoSim
--skills-count <N>int5Numero de especialidadesNao
--self-learningbooltrueAuto-update memoria (DR-0130)Nao

Exemplos praticos

Cenario 1: Adicionar expert em Reforma Tributaria

Contexto: Apos /forge MeuProjeto com modulo fiscal, voce descobre que precisa de conselho especialista em Reforma Tributaria 2026 (IBS/CBS).

/genius --name "Marcos Oliveira" --inspiration "Auditor fiscal senior, ex-receita federal" --domain "Fiscal" --skills-count 6

Output esperado:

[CREATE_GENIUS] Criando AG069...
  [+] .kilo/agents/genius-council/AG069-marcos-oliveira.md
  [+] .xforge/memory/genius/AG069/knowledge.jsonl (vazio)
  [+] 6 especialidades (IBS, CBS, ICMS-ST, partilha, DIFAL, Reforma 2026)
  [+] 5 perguntas caracteristicas
  [+] DR-0199-marcos-genius (status: draft)

[CONTINUIDADE]:
  - Use: /genius AG069 --topico "ICMS-ST para revenda SP 2026"
  - Self-learning ativado: AG069 memoria cresce com uso

Edge case: Se AG ID ja existe, wizard incrementa sufixo (Marcos Oliveira 2).

Cenario 2: Adicionar Grace Hopper (historica)

Contexto: Quer adicionar pioneira de compiladores para o Conselho.

/genius --name "Grace Hopper" --inspiration "Grace Hopper, COBOL, primeiro compilador A-0 (1952)" --domain "Linguagens"

Output esperado:

[CREATE_GENIUS] Criando AG067...
  [+] .kilo/agents/genius-council/AG067-grace-hopper.md
  [+] .xforge/memory/genius/AG067/knowledge.jsonl
  [+] 5 especialidades (Compiladores, COBOL, Debugging, Linguagens, Historia)
  [+] 5 perguntas: "O que Grace Hopper diria sobre..."
  [+] DR-0195-grace-hopper (status: draft)

Edge case: Genios historicos tem trustScore 100 (fonte canonica).

Cenario 3: Adicionar expert em investimentos (Warren Buffett)

Contexto: Projeto precisa de conselho sobre investimentos e valuation.

/genius --name "Warren Buffett" --inspiration "Warren Buffett, Berkshire Hathaway, value investing" --domain "Investimentos"

Output esperado:

[CREATE_GENIUS] Criando AG070...
  [+] .kilo/agents/genius-council/AG070-warren-buffett.md
  [+] 5 especialidades (Value Investing, Valuation, Risco, Portfolio, Macro)
  [+] DR-0200-warren-buffett (status: draft)

Edge case: Genio de negocio: use /genius AG070 --topico "avaliar oportunidade de investimento em X"
