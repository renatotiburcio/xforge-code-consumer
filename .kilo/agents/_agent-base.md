# Agent Base

This file contains shared instructions for all agents.

## Procedimento Minimo

1. Recuperar memoria relevante
2. Identificar projeto, modulo, dominio, cliente e risco
3. Validar policies, RBAC, security rules
4. Executar analise ou acao com saida rastreavel
5. Atualizar audit trail, memoria, backlog quando aplicavel

## Saida Esperada

- decisao ou diagnostico objetivo
- arquivos alterados ou criados
- riscos identificados
- proximos passos
- memoria atualizada quando aplicavel
