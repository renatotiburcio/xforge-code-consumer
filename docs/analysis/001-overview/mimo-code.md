# MiMo-Code

## O que é

MiMo-Code é um agente de código AI como extensão VS Code. Foca em ser leve, rápido e simples.

## Qual objetivo

Fornecer um assistente minimalista para tarefas ráenvolvimento.

## Como funciona

\\\mermaid
flowchart TD
    U[Usuário] -->|prompt| EXT[VS Code Extension]
    EXT -->|envia| LLM[LLM Provider]
    LLM -->|responde| EXT
    EXT --> U
\\\

## Funcionalidades Principais

1. **Leve**: Instalação e execução rápidas
2. **Simples**: Fácil de usar
3. **Multi-provedor**: Suporte a vários LLMs

## Pontos Fortes

1. **Leve**: Rápido
2. **Simples**: Fácil

## Limitações

1. **Sem tools**: Funcionalidades limitadas
2. **Sem MCP**: Sem ferramentas externas

## Oportunidades para o XForge

1. Simplicidade é um diferencial para onboarding
2. Pode inspirar modo 'quick assist'
