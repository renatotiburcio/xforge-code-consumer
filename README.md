# XForge Code Consumer

Template publico para criar projetos com XForge.

## Uso

```bash
git clone https://github.com/renatotiburcio/xforge-code-consumer.git meu-projeto
cd meu-projeto
xforge-init
```

## Comandos

- `/forge` - Desenvolvimento
- `/criar` - Criar artefatos
- `/desenvolver` - Executar tarefas
- `/testar` - Testes
- `/qualidade` - Quality gates
- `/release` - Releases
