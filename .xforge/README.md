# .xforge

Pasta persistente do projeto.

Guarda tudo que pertence ao projeto e não deve ser substituído automaticamente ao atualizar o Engineer:

- memória;
- conhecimento;
- project DNA;
- backlog;
- sprints;
- roadmap;
- changelog;
- relatórios;
- legados;
- incidentes;
- decisões;
- customizações;
- aprendizados.

Regra: `.xforge` nunca deve ser apagada em atualização.
