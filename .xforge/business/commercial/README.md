# Comercial / CRM

## Modulos principais
- **Leads**: captacao e qualificacao
- **Oportunidades**: pipeline de vendas
- **Pedidos**: orcamento -> pedido -> faturado
- **Comissoes**: calculo por vendedor/regiao
- **Pos-venda**: SAC, garantia, retorno

## Funil de vendas tipico
1. **Lead** (topo): captado de marketing
2. **Qualificado (MQL)**: tem fit + interesse
3. **Oportunidade (SQL)**: tem orcamento + autoridade
4. **Proposta**: orcamento formal enviado
5. **Negociação**: revisao de termos
6. **Ganho/Perdido**: fechamento
