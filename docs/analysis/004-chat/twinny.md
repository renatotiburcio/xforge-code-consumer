# Twinny — Sistema de Chat

## Arquitetura

O chat do Twinny é simples:

1. Sidebar com campo de entrada
2. Ollama processa prompt localmente
3. Resposta exibida na webview

## Funcionalidades

1. Chat básico
2. Histórico local
3. Local-first (sem cloud)

## Stack

| Tecnologia | Versão |
|------------|--------|
| React | latest |
| TypeScript | 5.x |
| Ollama | latest |

## Pontos Fortes

1. Privacidade
2. Simplicidade

## Limitações

1. Sem streaming
2. Sem @context
3. Sem multi-sessão
