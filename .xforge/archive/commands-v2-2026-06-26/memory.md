# /memory

Gerencie memoria e conhecimento.

## Sintaxe

```
/memory status
/memory recover
/memory learn
/memory cycle
/memory ttl
```

## Sub-comandos

| Sub-comando | Descricao | Substitui |
|-------------|-----------|-----------|
| status | Status da memoria | memory-sharding-analysis |
| recover | Recuperar contexto | memory-recover-context |
| learn | Aprender com feedback | memory-auto-learn, memory-auto-cycle, aprender |
| cycle | Ciclo automatico | autonomous-learning-loop |
| ttl | Validacao TTL | memory-ttl-validation |
| mcp | Teste MCP | memory-mcp-test |
