Skills - Catalogo (172 skills)

O que e

Catalogo de todas as 172 skills do XForge, organizadas em 12 subcategorias.

Quando usar

  • Quando voce quer descobrir qual skill atende sua necessidade
  • Quando quer explorar um dominio especifico (Backend, Fiscal, LGPD, etc)
  • Quando quer ver a cobertura do template por area

Como usar

Esta pagina e' o indice. Clique em uma subcategoria para ver a lista detalhada de skills com descricoes.

Parametros

CampoDescricao
Total172 skills distribuidas em 12 subcategorias
Tamanho medio~80 linhas por SKILL.md
Origem.kilo/skills/<nome>/SKILL.md

Exemplos praticos

Cenario 1: Encontrar skill por dominio

Contexto: voce sabe o dominio mas nao a skill especifica.

# Clique na subcategoria relevante abaixo
# Role ate a skill especifica

Cenario 2: Achar skill canonica XForge

Contexto: voce quer skills marcadas como canonicas.

# Procure por "xforge" no nome da skill

Cenario 3: Ver cobertura por stack

Contexto: voce quer saber quantas skills cobrem seu stack.

# Backend = .NET, Go, Rust, JVM
# Frontend = React, Vue, Angular, Svelte, Next

Troubleshooting

Skill nao encontrada em nenhuma categoria

Causa: skill descontinuada ou renomeada.

Solucao: verifique ls .kilo/skills/.

Skill em 2 subcategorias

Causa: heuristica de categorizacao.

Solucao: edite o frontmatter category: no SKILL.md.
